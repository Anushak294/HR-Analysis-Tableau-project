# HR Analysis Tableau project



Project Overview

This project focuses on analyzing employee attrition, workforce trends, job satisfaction, and department performance using Tableau. The dashboard provides interactive visualizations and key insights to help understand employee behavior and organizational trends.







Objectives

\- Analyze employee attrition patterns

\- Identify department-wise employee turnover

\- Study age group distribution and trends

\- Evaluate job satisfaction levels

\- Generate business insights using data visualization





&#x20;Dashboard Features

\- Employee Attrition Analysis

\- Department-wise Attrition Insights

\- Age Group Analysis

\- Gender-based Employee Analysis

\- Job Satisfaction Dashboard

\- KPI Metrics and Interactive Filters







Tools \& Technologies Used

\- Tableau

\- Microsoft Excel



\---



Dataset Information

The dataset contains HR employee information including:

\- Employee ID

\- Age

\- Gender

\- Department

\- Education Field

\- Job Satisfaction

\- Attrition Status

\- Salary Details

\- Work Experience







Key Insights

\- Higher attrition was observed among younger employees

\- Certain departments experienced increased employee turnover

\- Job satisfaction had a strong impact on employee retention

\- Workforce trends were effectively analyzed using interactive dashboards





Conclusion

This project demonstrates the use of Tableau for HR analytics and business intelligence. It helps in understanding employee trends, improving retention strategies, and supporting data-driven decision-making.
